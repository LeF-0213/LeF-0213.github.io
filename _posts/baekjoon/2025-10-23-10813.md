## Python
```python
import sys

n, m = map(int, sys.stdin.readline().split())
li = [i for i in range(1, n + 1)]

for _ in range(m):
    i, j = map(int, sys.stdin.readline().split())
    li[i - 1], li[j - 1] = li[j - 1], li[i - 1]
    
print(*li)
```
