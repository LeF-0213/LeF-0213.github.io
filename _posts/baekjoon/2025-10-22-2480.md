## Python
```python
a, b, c = map(int, input().split())
li = [a, b, c]

if(a == b and b == c):
    print(10000 + a * 1000)
elif(a == b or b == c or c == a):
    n = a if (a == b) or (a == c) else b
    print(1000 + n * 100)
else:
    print(max(li) * 100)
```
