## Python

```python
s = input().upper()
dial = 0

for i in s:
    idx = ord(i) - 65           # 대문자라 'A' - 65 = 0
    if(idx >= 18 and idx < 25): # 7번 다이얼이 PQRS라 S 이후부터 idx -1
        idx -= 1
    elif(idx == 25): # 9번 다이얼이 WXYZ라 Z 인덱스 값 -2
        idx -=2
    dial += (idx + 6) // 3 + 1 # 한 번 돌릴때마다 1초씩 추가
    
print(dial)
```
