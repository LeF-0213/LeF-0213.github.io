```python

    def add(self, num):
        return self.result + num

cal = Calculator()

# 일반적인 호출 (권장)
cal.add(3)

# 실제로는 아래처럼 동작함
Calculator.add(cal, 3)

cal.add(cal, 3)
# Calculator.add() takes 2 positional arguments but 3 were given

Calculator.add(cal, cal, 3)
```
