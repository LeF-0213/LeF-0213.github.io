# 파일 입출력
파이썬에서 파일 입출력은 다양한 용도로 사용됩니다. 텍스트 파일, 바이너리 파일, CSV 파일, JSON 파일 등을 읽고 쓸 수 있습니다.

## 파일 열기
파일을 열려면 open 함수를 사용한다.
`open` 함수는 존재하는 파일을 쓰기도 하고, 읽어오기도 하는 함수이다.

```python
f = open("파일경로", "모드")
```

| 모드 | 설명 |
|:-------------:|:--------------------------------------------------:|
| `r` / `rt` | 현재 파일을 **읽기 모드**로 염 (기본값, 파일이 존재해야 함) |
| `w` / `wt` | 파일을 **쓰기 모드**로 염 (기존 파일이 있으면 덮어쓰기) |
| `a` / `at` | 파일을 **추가 모드**로 염 (파일 끝에 내용이 이어서 추가됨) |
| `rb` | **바이너리 읽기 모드**, 이미지·오디오 등 이진 데이터 읽을 때 사용 |
| `wb` | **바이너리 쓰기 모드**, 기존 내용을 지우고 새로 작성 |
| `ab` | **바이너리 추가 모드**, 기존 파일 뒤에 이진 데이터 추가 |
| `r+` | **읽기/쓰기 겸용 모드**, 파일이 반드시 존재해야 함 |
| `w+` | **읽기/쓰기 겸용 모드**, 기존 파일을 덮어씀 |
| `a+` | **읽기/쓰기 겸용 추가 모드**, 파일이 없으면 새로 생성 |


## 파일 쓰기
### wirte()
문자열을 파일에 쓰는 메서드

```python
f = open('example.txt', 'w')
f.write('Hello, Python!\n')
f.close()
```
### writelines()
문자열 리스트를 파일에 쓰는 메서드

```python
f = open('example.txt', 'w')
f.writelines(['Line 1\n', 'Line 2\n'])
f.close()
```
## 파일 경로
파일 경로는 파일의 위치를 나타내는 문자열입니다. 이 경로를 통해 파이썬은 파일을 찾아서 해당 파일을 읽거나 쓸 수 있습니다. 파일 경로는 크게 상대 경로와 절대 경로 두 가지로 구분됩니다.

### 상대 경로(Relative Path)
현재 작업 디렉토리에 대한 파일의 위치를 나타낸다.

| 경로 표현 | 설명 |
|:---------:|--------------------------------|
| `"data.txt"` | 현재 디렉토리의 **data.txt** 파일 |
| `"./data.txt"` | 현재 디렉토리의 **data.txt** 파일 (상동, `./`은 현재 경로 의미) |
| `"subfolder/data.txt"` | 현재 디렉토리의 하위 폴더 **subfolder** 안의 **data.txt** 파일 |
| `"../siblingfolder/data.txt"` | 현재 디렉토리와 동일한 상위 경로에 있는 **siblingfolder** 폴더 안의 **data.txt** 파일 |

### 절대 경로(Absolute Path)
파일 시스템의 루트로부터 전체 경로를 나타낸다.

| 운영체제            | 예시 경로                                  |
| :-------------- | :------------------------------------- |
| **Windows**     | `C:\Users\UserName\Documents\data.txt` |
| **Linux / Mac** | `/home/username/Documents/data.txt`    |


## with문 사용하기
with 문은 자원(리소스)을 열고 자동으로 정리(닫기)까지 해주는 문법입니다. 파일 입출력, 데이터베이스 연결, 락(lock) 사용 등 열고 닫아야 하는 작업에 주로 사용됩니다. 예를 들어 파일을 열 때 open()을 사용하고, 나중에 close()로 닫아야 합니다. 그런데 중간에 에러가 나면 close()가 실행되지 않습니다. 이때 with 문을 사용하면, 자동으로 닫아줍니다.

```python
with 열기_함수 as 변수:
    # 이 안에서 자원을 사용
```

```python
with open('./data/word.txt', 'w') as f:
  while True:
    data = input('단어를 입력하세요: ')
    if data.lower == 'quit':
      break
    f.write(data + '\n')
```
> with 문은 \_\_enter\_\_()와 \_\_exit\_\_() 메서드를 가진 객체와 함께 사용됩니다. 이걸 **컨텍스트 매니저(context manager)**라고 합니다.

```python
class MyContext:
    def __enter__(self):
        print("시작합니다!")
        return "리소스"

    def __exit__(self, exc_type, exc_value, traceback):
        print("끝났습니다!")

with MyContext() as resource:
    print(f"{resource}를 사용 중입니다")
```

## 파일 읽기
### read()
파일의 모든 내용을 문자열로 반환
#### read()에 글자수를 지정해주지 않을 때

```python
f = open('example.txt', 'r')
content = f.read()
print(content)
f.close()

# 결과
Hello, Python!
Line 1
Line 2
```
#### read()에 글자수를 지정해 줄 때

```python
f = open('example.txt', 'r')
content = f.read(10)
print(content)
f.close()

# 결과
Hello, Pyt
```
#### read()를 while문 안에 돌릴 떼(글자수 지정해줘도 값은 같음)

```python
file = open('./data/word.txt', 'rt')
while True:
  data = file.read(3)
  if not data:
    break
  print(data, end='')

# 결과
apple
banana
orange
melon
```
