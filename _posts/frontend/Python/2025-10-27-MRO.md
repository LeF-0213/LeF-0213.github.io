# MRO(Mehtod Resolution Order)란?


다이아몬드 상속 문제 해결
```python
class Animal:
    def __init__(self):
        print("Animal 초기화")
    
    def speak(self):
        return "소리"

class Mammal(Animal):
    def __init__(self):
        super().__init__()
        print("Mammal 초기화")
    
    def speak(self):
        return "포유류 소리"

class WingedAnimal(Animal):
    def __init__(self):
        super().__init__()
        print("WingedAnimal 초기화")
    
    def speak(self):
        return "날개 동물 소리"

class Bat(Mammal, WingedAnimal):
    def __init__(self):
        super().__init__()
        print("Bat 초기화")

print("MRO:", [cls.__name__ for cls in Bat.__mro__])
bat = Bat()
print(bat.speak())
```

**출력:**
```
MRO: ['Bat', 'Mammal', 'WingedAnimal', 'Animal', 'object']
Animal 초기화
WingedAnimal 초기화
Mammal 초기화
Bat 초기화
포유류 소리
```
