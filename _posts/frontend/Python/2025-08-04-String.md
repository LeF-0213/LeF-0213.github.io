# 문자열
문자열을 표현할 때  `""`을 사용한다.    
파이썬에서는 문자열을 나타내는데 `''`, `""` 둘 중 어는 것을 사용할 수 있다.   
`print()`에 대한 출력은 `''`로 표시한다.

## 아스키코드(ASCII) 와 유니코드(Unicode)
### 아스키
* American Standard Code for Information Interchange의 약자로, 미국에서 개발된 문자 인코딩 표준이다. 
* 아스키 코드는 7비트로 구성되어 총 128개의 문자(0~127)를 표현한다.

### 유니코드
* 전 세계의 다양한 문자와 기호를 포함하기 위해 개발된 표준이다. 
* 유니코드는 각 문자를 고유한 코드 포인트로 표현하며, 다양한 인코딩 방식(UFT-8, UTF-16, UTF-32 등)을 지원한다.

# 문자열 함수와 연산자
## 문자열 길이 확인하기
```python
text = 'Hello, Python!'
len(text) # 14
```
## 문자열 합치기
```python
name = '김사과'
age = 20
message = name + ", " + str(age) + "살"
print(message) # 김사과, 20살
```
## 문자열 반복하기
```python
apple = '🍎' * 10
print(apple) # 🍎🍎🍎🍎🍎🍎🍎🍎🍎🍎
```
## 문자열 인덱싱
```python
s = "문자열 인덱싱"
print(s[0]) # 문
print(s[-1]) # 싱
```
## 문자열 슬라이싱
```python
s = "문자열 슬라이싱"
print(s[0:4]) # 문자열
print(s[4:]) # 슬라이싱
print(s[:4]) # 문자열
```

# 문자열 메서드
## 대소문자 관련 함수
### upper()
모든 문자를 대문자로 변환
```python
text = "hello"
print(text.upper())
# HELLO
```

### lower()
모든 문자를 소문자로 변환
```python
text = "HELLO"
print(text.lower())
# hello
```

### title()
각 단어의 첫 글자를 대문자로 변환
```python
text = "hello world"
print(text.title())
# Hello World
```

### swapcase()
대문자는 소문자로, 소문자는 대문자로 변환
```python
text = "Hello World"
print(text.swapcase())
# hELLO wORLD
```

## 특정 문자열의 등장 횟수 확인
```python
text = 'Hello, Python!'
print(text.count('l'))
print(text.count('ll'))
```

## 특정 부분 문자열의 첫 번째 등장 위치(인덱스) 확인
```python
text = 'Hello, Python!'
print(text.find('l'))     # 2
print(text.find('ll'))    # 2
print(text.find('g'))     # -1
print(text.rfind('l'))    # 3
print(text.find('o', 5))  # 11
```

## 문자열 대체
```python
text = 'Hello, Python!'
new_text = text.replace('Python', 'World')
print(new_text) # Hello, World!
```

## 공백 삭제 함수
### strip()
양쪽(앞/뒤) 공백 제거
```python
text = "   Hello   "
print(text.strip())
# Hello
```

### rstrip()
오른쪽 공백 제거
```python
text = "Hello   "
print(text.rstrip())
# Hello
```

### lstrip()
왼쪽 공백 제거
```python
text = "   Hello"
print(text.lstrip())
# Hello
```

## 문자열 분리
### split()
지정한 구분자로 문자열을 나눠 리스트로 반환
```python
text = "apple,banana,grape"
print(text.split(','))
# ['apple', 'banana', 'grape']
```

### splitlines()
줄바꿈 문자 기준으로 나눔
```python
text = "hello\nworld"
print(text.splitlines())
# ['hello', 'world']
```
* csv: ','로 나누어진 문서

## 문자열 결합
### join()
리스트의 문자열을 하나로 연결
```python
city = "서울시"
street = "역삼로"

address = ' '.join((city, street))
print('주소: ', address)  # 주소:  서울시 역삼로
```

## 접두사와 접미사 확인
```python
text = 'Hello, Python!'
starts_with_hello = text.startswith('Hello')
print(starts_with_hello)  # True
ends_with_world = text.endswith('World!')
print(ends_with_world)    # False
```

## map()
리스트의 각 요소에 함수를 적용(리스트 값을 하나 하나씩 함수명에 대입)
```python
nums = ['10', '20', '30']
int_nums = list(map(int, nums))
print(int_nums) 
# [10, 20, 30]

words = ['apple', 'banana', 'grape']
upper_words = list(map(str.upper, words))
print(upper_words)  
# ['APPLE', 'BANANA', 'GRAPE']
```

## 문자열은 불변(Immutable) 데이터 타입
* 문자열 객체가 생성된 후에는 그 값을 바꿀 수 없다는 것을 의미한다.
* 예를 들어, 한 번 생성된 문자열 객체 내에서 특정 문자를 바꿀 수 없다. 
* 만약 문자열의 일부를 변경하고 싶다면, 새로운 문자열을 만들어서 교체해야 한다.

```python
my_string = 'Hello'
print(my_string[0]) # H

my_string[0] = 'h' # 오류 발생
# TypeError: 'str' object does not support item assignment

new_string = 'h' + my_string[1:]
print(new_string) # hello
```

# Python 대소문자 변환으로 보는 메모리 동작 과정
![UpperCaseMemoryImg](https://github.com/user-attachments/assets/e03e9935-48d7-4a8f-b45e-fc0b95949a2d)
## 🔑 핵심 포인트
### 문자열 불변성 (Immutability)
* **원본은 절대 변하지 않음**: `"Hello, Python!"`은 그대로 유지
* **새 객체 생성**: `upper()`는 새로운 대문자 문자열 생성
* **임시 반환**: `새 객체는 출력 후 사라짐` (재할당 안 하면)

### 메모리 변화 과정
* **원본**: text → "Hello, Python!"
* **upper() 실행**: 새 객체 "HELLO, PYTHON!" 생성
* **출력 후**: `임시 객체 소멸`, `text는 여전히 원본 가리킴`

### 활용 예시
```txt
다음과 같이 출력될 수 있도록 코딩하시오   
출력 예]    
문자열을 입력하세요 : Java    
입력된 문자열을 역으로 출력 : avaJ
```
```python
str = input("문자열을 입력하세요 : ")

print('입력된 문자열을 역으로 출력 : ', end = "")
for i in range (len(str) - 1, -1, -1):
    print(str[i], end = '')

print('입력된 문자열을 역으로 출력 :', str[::-1])
```

### 활용 예시
```txt
문자열을 입력하시오.
입력한 문자열 중 알파벳 'o'를 만나면 '*'로 변경하는 프로그램을 작성하시오.

입력 예 ] 문자열을 입력하세요. Programming    
출력 예 ] 문자열은 : Pr*gramming    
```
```python
str = input("문자열을 입력하세요. : ")
print('입력된 문자열은 : ', end ="")
        
for i in range(0, len(str)):
    if(str[i].lower() == "o"):
        print("*", end="");
    else:
        print(str[i], end = "");
```
