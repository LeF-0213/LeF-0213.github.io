### 📊 정리 비교표

| 항목                 | 저장 위치    | 지속 시간          | 접근 가능 언어           | 특징            |
| ------------------ | -------- | -------------- | ------------------ | ------------- |
| **sessionStorage** | 메모리(RAM) | 탭 닫을 때까지       | ✅ 자바스크립트           | 세션 단위, 임시 저장  |
| **localStorage**   | 디스크      | 브라우저 종료 후에도 유지 | ✅ 자바스크립트           | 반영구 저장        |
| **cookie**         | 디스크      | 설정된 만료일까지      | ✅ JS / 서버(HTTP 헤더) | 서버와 자동 동기화 가능 |
