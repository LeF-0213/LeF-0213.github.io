# CSS란?

> CSS(Cascading Style Sheets)는 HTML과 함께 사용되어 웹 페이지의 스타일과 레이아웃을 정의하는 스타일시트 언어이다.  
> HTML이 웹 페이지의 구조를 담당한다면, CSS는 글꼴, 색상, 여백, 정렬, 배경 등 시각적인 요소들을 제어하여 웹 페이지를 보다 보기 좋고 사용하기 쉽게 만들어준다.  
> 또한, CSS는 동일한 스타일을 여러 페이지에 일관되게 적용할 수 있게 해주며, 미디어 쿼리를 사용하면 화면 크기나 장치 종류에 따라 반응형 디자인도 구현할 수 있다.

## CSS 문법

```python
선택자 {
  속성 : 값;
}
```

### 선택자

> 선택자는 "어떤 HTML 요소에 스타일을 적용할지"를 지정한다.  
> 예를 들어, <p> 태그에 스타일을 적용하고 싶다면 p라는 선택자를 사용한다.

### 속성(Property)과 값(Value)

중괄호 {}안에는 스타일을 정의하는 속성과 값이 들어간다.

> **속성(Property)**: 변경하고 싶은 스타일의 종류이다.  
> ex: color, font_size, background-color, margin, padding 등  
> **값(Value)**: 해당 속성에 어떤 값을 적용할지 정한다.  
> ex: blue, 16px, center, 10px 등

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>css란</title>
  <style>
    body { 
      background-color: deepskyblue;
    }

    h1 {
      color: deeppink; 
      text-align: center;
    }
  </style>
</head>
<body>
  <h1>CSS란?</h1>
</body>
</html>
```

## 스타일 적용 방법

### 인라인(inline) 스타일

> HTML 태그 안에 직접 style 속성으로 CSS를 작성하는 방식이다. 해당 태그 하나에만 스타일이 적용된다.  
> **장점**: 빠르게 특정 요소에 스타일 적용 가능  
> **단점**: 유지보수가 어려움, 스타일 중복이 많아짐

```html
<p style="color: deeppink; font-style: italic">
  이 문장은 인라인 스타일로 스타일이 적용되었습니다.
</p>
```
### 내부(internal) 스타일시트

> HTML 문서의 <head>태그 안에 <style> 태그를 사용하여 CSS를 작성하는 방식이다. 해당 HTML 파일 전체에 적용할 수 있다.  
> **장점**: 한 문서 안에서 스타일을 통일되게 관리 가능  
> **단점**: HTML과 CSS가 섞여 있어 코드가 복잡해질 수 있다.

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>스타일 적용방법</title>
    <style>
      p#internal {
        color: deepskyblue;
        font-weight: bold;
      }
    </style>
  </head>
  <body>
    <!-- 내부 스타일 적용 -->
    <p id="internal">이 문장은 내부 스타일로 스타일이 적용되었습니다.</p>
  </body>
</html>
```
### 외부(external) 스타일 시트

> CSS 코드를 별도의 .css 파일에 작성하고, HTML 문서에서 link 태그로 연결하는 방식이다. 여려 HTML 파일에서 **재사용** 가능하며 가장 많이 사용되는 방식이다.        
> **장점**: 코드 분리로 관리가 쉬움, 여러 페이지에서 재사용 가능      
> **단점**: 파일이 분리되어 있으므로 연결이 필요함

```css
p#external {
  color: gold;
  text-align: center;
}
```

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>스타일 적용방법</title>
    <!-- link rel="stylesheet" -->
    <link rel="stylesheet" href="./css/external.css">
  </head>
  <body>
    <!-- 외부 스타일 적용 -->
    <p id="external">이 문장은 외부 스타일로 스타일이 적용되었습니다.</p>
  </body>
</html>
```

### <link rel="stylesheet">

> <link rel="stylesheet">는 HTML 문서에서 외부 CSS 파일을 연결할 때 사용하는 태그이다. 여기서 link는 HTML과 외부 리소스를 연결(link)한다는 의미이고, rel="stylesheet"는 연결된 파일의 종류가 스타일시트(CSS)임을 나타낸다. 이 태그는 <head> 태그 안에 위치하며, href 속성에는 연결할 CSS 파일의 경로를 지정한다.      
> 예를 들어 <link rel="stylesheet" href="style.css">는 style.css 파일을 HTML 문서에 적용하여, 그 안의 스타일을 HTML 요소에 반영하게 한다.

## 다양한 선택자

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>선택자</title>
    <style>
      /* 태그 선택자 */
      h1 {
        color: darkolivegreen;
        text-align: center;
      }
      /* id 선택자 */
      #student-list {
        background-color: #b4a8a8;
        padding: 30px;
      }
      /* 클래스 선택자 */
      .highlight {
        font-size: 14px;
        font-weight: bold;
      }
      /* 그룹 선택자 */
      p,
      li {
        background-color: burlywood;
      }
      /* 속성 선택자 */
      a[href='https://www.naver.com']
      {
        color: red;
      }
      /* 자식 선택자 */
      #student-list > li {
        list-style-type: square;
      }
      /* 자손 선택자(자식 포함) */
      #student-list p {
        color: gray;
        font-style: italic;
      }
      /* 형제 선택자 */
      #selector + p {
        /* 인접 */
        background-color: blanchedalmond;
      }
      #selector ~ p {
        /* 일반 */
        font-size: 30px;
      }
      /* 의사 클래스 선택자 */
      a:link {
        text-decoration: none;
      }
      a:visited {
        color: cadetblue;
        text-decoration: none;
      }
      a:hover {
        text-decoration: underline;
      }
      a:active {
        color: red;
      }
    </style>
  </head>
  <body>
    <h1>선택자</h1>
    <p class="highlight" id="selector">이 문단은 첫 번째 문단입니다</p>
    <p>이 문단은 두 번째 문단입니다</p>
    <span class="highlight">이 문장은 첫 번째 문장입니다.</span>
    <hr />

    <ul id="student-list">
      <li>
        김사과
        <p>안녕</p>
      </li>
      <li>반하나</li>
      <li>오렌지</li>
      <p>출석부</p>
    </ul>

    <hr />
    <p><a href="https://www.naver.com">네이버</a></p>
    <p><a href="https://www.daum.com">다음</a></p>
    <p><a href="https://www.google.com">구글</a></p>
  </body>
</html>
```
