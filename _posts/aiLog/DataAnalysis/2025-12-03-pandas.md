# Pandas(판다스)

판다스(Pandas)는 데이터 분석을 위한 파이썬 라이브러리 중 하나로, 표 형태의 데이터나 다양한 형태의 데이터를 쉽게 처리하고 분석할 수 있도록 도와주는 도구이다. 주로 데이터프레임(DataFrame)이라는 자료구조를 제공하며, 이를 통해 테이블 형태의 데이터를 다루기 용이하다.

```bash

pip install pandas
```


## Series와 DataFrame

**🖍️ Tip! Series와 DataFrame의 구성요소를 잘 알아야 한다.**

### Series

Series는 1차원 배열과 같은 자료구조로 하나의 열을 나타낸다. 또한 각 요소는 인덱스(index)와 값(value)로 구성되어 있다. 값은 Numpy의 ndarray 기반으로 저장된다. Series는 다양한 데이터 타입을 가질 수 있으며 정수, 실수, 문자열 등 다양한 형태의 데이터를 담을 수 있다.

```python
idx = ['김사과', '반하나', '오렌지', '이메론', '배애리']
data = [67, 75, 90, 62, 98]

pd.Series(data)
```


### DataFrame

`데이터프레임(DataFrame)`은 판다스(Pandas) 라이브러리에서 제공하는 중요하고 강력한 데이터 구조로, **2차원의 테이블 형태 데이터를 다루는데 사용**된다.
또한 각 요소는 인덱스(index), 열(column), 값(value)로 구성되어 있다. 데이터프레임은 행과 열로 이루어져 있으며, 각 열은 다양한 데이터 타입을 가질 수 있다.
값은 Numpy의 ndarray 기반으로 저장된다. 
즉, 데이터프레임은 Series의 결합으로 이루어져있다고 볼 수 있다.

```python
data = [[67, 93, 91],
        [75, 68, 96],
        [87, 81, 82],
        [62, 70, 75],
        [98, 56, 87]]

idx = ['김사과', '반하나', '오렌지', '이메론', '배애리']
col = ['국어', '영어', '수학']
```

```python
pd.DataFrame(data)
pd.DataFrame(data, idx)
```

## 통계 함수

### ※ 평균과 중앙값
평균은 모든 데이터를 더한 후, 데이터 개수로 나눈 값이다. 데이터를 고르게 분배했을 때, 한 데이터가 가질 수 있는 이론적인 중심값을 의미한다. 
중앙값은 데이터를 크기 순서대로 정렬했을 때, 가운데 위치하는 값이다. 데이터의 순서에만 영향을 받고, 값의 크기에는 영향을 받지 않는다.
