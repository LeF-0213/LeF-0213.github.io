# Crawling(크롤링)

크롤링(Crawling)이란 웹 크롤러(Web Crawler) 또는 스크레이퍼(Scraper)라고 불리는 프로그램이나 스크립트를 사용하여 인터넷상의 웹 페이지에서 데이터를 자동으로 수집하는 과정을 말한다.

```python
import requests

site = '크롤링한 사이트 주소'
request = requests.get(site)
print(request) # <Response [200]>
```

### import requests
* **개념**: Python에서 **HTTP 요청**을 보낼 수 있게 해주는 라이브러리이다.
* **역할**: 웹 페이지의 HTML 내용을 가져오기 위해 서버에 **GET 요청**을 보내는데 사용된다.
* **`requests` 모듈**: 웹 페이지의 주소(URL)을 입력받아 서버와 통신하고, 응답(Response) 객체를 돌려준다.

### requests.get()

지정된 URL로 **HTTP GET 요청**을 보낸다. 웹 브라우저가 웹 페이지를 띄우기 위해 서버에 요청하는 것과 동일한 역할을 프로그래밍 방식으로 수행하며, 서버로부터 해당 페이지의 **HTML, JSON 등의 데이터**를 응답으로 받아온다.

**기본 구문**

```python
requests.get(url, params=None, headers=None, timeout=None, **kwargs)
```

**주요 인자(Key Parameters)**

| **인자명** | **역할** | **상세 설명** |
|:--------------:|----------|----------------|
| **url(필수)** | 요청을 보낼 웹 주소 | 요청을 보낼 웹 페이지 또는 API의 URL (문자열) |
| **params** | 쿼리 파라미터 (Query Parameters) | URL 뒤에 붙는 `?key1=value1&key2=value2` 형태의 데이터를 딕셔너리로 전달 |
| **headers** | HTTP 헤더 | 요청에 포함될 부가적인 헤더 정보 (예: User-Agent, Content-Type) 를 딕셔너리 형태로 전달 |
| **timeout** | 시간 제한 | 서버 응답을 기다리는 최대 시간(초). 초과하면 예외 발생 |

```python
from bs4 import BeautifulSoup

soup = BeautifulSoup(request.text, 'html.parser')

box_div = soup.find('div', {'class': 'thrv-columns'})
```

#### bs4

**Beautiful Soup 4**의 약자로, 웹 크롤링 시 가장 널리 사용되는 Python 라이브러리 중 하나이다. `requests`로 가져온 HTML을 **파싱(Parsing)**하여 프로그래밍 방식으로 데이터를 검색하고 조작할 수 있게 도와준다.

#### BeautifulSoup

`bs4` 라이브러리 내의 **주요 클래스**이다. HTML 문서를 Python 객체로 변환하는 역할을 한다.

**주요 인자(Key Parameters)**

```python
soup = BeautifulSoup('파싱할 문서', '파서 지정')
```

| 인자 위치 | 역할 | 상세 설명 |
|:------:|
| 첫번째 인자 | 파싱할 문서 | 웹 서버에서 받은 HTML 또는 XML 소스 코드 문자열 |
| 두번째 인자 | 파서(Parser) 선택 | HTML을 해석하는 엔진을 지정한다. |
