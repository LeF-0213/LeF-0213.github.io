## Java
```java
class Solution {
    public int solution(int n) {
        int answer = 0;
        int start = 1, end = 1;
        int sum = 0;
        
        while(start <= n / 2) {
            if(sum < n) {
                sum += end;
                end++;
            } else if(sum > n) {
                sum -= start;
                start++;
            } else {
                answer++;
                sum -= start;
                start++;
            }
        }
        
        return answer + 1;
    }
}
```
