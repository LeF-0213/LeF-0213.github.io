## Java
```java
class Solution {
    boolean solution(String s) {
        int cnt = 0;
        
        for(int i = 0; i < s.length(); i++) {
            if(s.charAt(i) == '(') {
                cnt++;
            } else {
                cnt--;
                if(cnt < 0) {
                    return false;
                }
            }
        }
        
        return cnt == 0;
    }
}
```

## Python
```python
def solution(s):
    stack = []
    
    for i in s:
        if (i == "("):
            stack.append(i)
        else:
            if not stack:
                return False
            stack.pop()

    return not stack
```
