## Java
```java
class Solution {
    public String solution(String s) {
        String[] stringArr = s.split(" ");
        int max = Integer.parseInt(stringArr[0]);
        int min = Integer.parseInt(stringArr[0]);
        
        for(int i = 0; i < stringArr.length; i++) {
            int n = Integer.parseInt(stringArr[i]);
            max = Math.max(max, n);
            min = Math.min(min, n);
        }
        
        return min + " " + max;
    }
}
```
