## Java
```java
class Solution {
    public int solution(int n) {
        int ans = 0;
        int f0 = 0, f1 = 1;
        
        for(int i = 2; i <= n; i++) {
            ans = (f0 + f1) % 1234567;
            f0 = f1;
            f1 = ans;
        }
        
        return ans;
    }
}
```
