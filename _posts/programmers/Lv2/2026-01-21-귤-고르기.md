## Python

```python
def solution(k, tangerine):
    answer = 0
    kind = {}
    
    for i in tangerine:
        kind[i] = kind.get(i, 0) + 1
        
    cnt = 0
    for i in sorted(kind.values(), reverse=True):
        if (cnt < k):
            cnt += i
            answer += 1
            
    return answer
```
