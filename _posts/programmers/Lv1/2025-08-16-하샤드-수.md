## Java
```java
class Solution {
    public boolean solution(int x) {
        boolean answer = true;
        int sum = 0;
        int copyX = x;
        
        while(x >= 1) {
            sum += x % 10;
            x /= 10;
        }
    
        return answer = copyX % sum == 0;
    }
}
```
