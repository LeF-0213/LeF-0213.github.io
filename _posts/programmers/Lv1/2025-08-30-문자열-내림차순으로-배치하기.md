## Java
```java
import java.util.Arrays;

class Solution {
    public String solution(String s) {
        String answer = "";
        char[] ch = s.toCharArray();
        Arrays.sort(ch);
        return answer = new StringBuilder(new String(ch)).reverse().toString();
    }
}
```
