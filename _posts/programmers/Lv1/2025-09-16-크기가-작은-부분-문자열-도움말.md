## Java
```java
class Solution {
    public int solution(String t, String p) {
        int answer = 0;
        int lenT = t.length();
        int lenP = p.length();
        Long P = Long.parseLong(p);
        
        for(int i = 0; i <= lenT - lenP; i++) {
            Long num = Long.parseLong(t.substring(i, i + lenP));
            if(num <= P) {
                answer++;
            }
        }
        
        return answer;
    }
}
```
