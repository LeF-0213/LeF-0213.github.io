## Java
```java
class Solution {
    boolean solution(String s) {
        boolean answer = true;
        char[] ar = s.toUpperCase().toCharArray();
        int numP = 0;
        int numY = 0;
        
        for(char c : ar) {
            if(c == 'P') {
                numP++;
            } else if(c == 'Y') {
                numY++;
            }
        }

        return answer = numP == numY;
    }
}
```

## Python
```python
from collections import Counter
def solution(s):
    c = Counter(s.lower()) 
    return c['p'] == c['y']
```
