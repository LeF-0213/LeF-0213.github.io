## Java
```java
class Solution {
    public long solution(int a, int b) {
        long answer = 0;
        int max = 0;
        int min = 0;
        
        max = Math.max(a, b);
        min = Math.min(a, b);
        for(int i = min; i <= max; i++) {
            answer += i;
        }
        
        return answer;
    }
}
```

## Python
```python
def solution(a, b):
    if(a > b):
        a, b = b, a
    
    return sum(range(a, b + 1))
```
