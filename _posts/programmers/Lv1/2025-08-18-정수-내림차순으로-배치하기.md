## Java
```java
class Solution {
    public long solution(long n) {
        long answer = 0;
        
        char[] list = String.valueOf(n).toCharArray();
        Arrays.sort(list);
        
        StringBuilder sb = new StringBuilder();
        sb.append(list);
        
        return answer = Long.parseLong(sb.reverse().toString());
    }
}
```
