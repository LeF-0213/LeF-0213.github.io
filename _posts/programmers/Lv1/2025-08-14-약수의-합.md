## Java
```java
class Solution {
    public int solution(int n) {
        int answer = 0;
        
        for(int i = 1; i <= n / 2; i++) {
            if(n % i == 0) {
                answer += i;
            }
        }
        
        answer += n;
        return answer;
    }
}
```

## Python
```python
def solution(n):
    answer = sum([i for i in range(1, n // 2 + 1) if (n % i == 0)]) + n
    return answer
```
