## Java
```java
class Solution {
    public int solution(String s) {
        String[] str = {"zero", "one", "two", "three", "four", "five", "six", "seven", "eight", "nine"};
        
        for(int i = 0; i < 10; i++) {
            s = s.replace(str[i], String.valueOf(i));
        }
        
        return Integer.parseInt(s);
    }
}
```
