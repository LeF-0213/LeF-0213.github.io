## Java
```java
class Solution {
    public int solution(int[] absolutes, boolean[] signs) {
        int answer = 0;
        
        for(int i = 0; i < signs.length; i++) {
            if(signs[i]) {
                answer += absolutes[i];
            } else {
                answer -= absolutes[i];
            }
        }
        
        return answer;
    }
}
```

## Python
```python
def solution(absolutes, signs):
    ans = 0
    
    for ab, s in zip(absolutes, signs):
        if(s):
            ans += ab
        else:
            ans -= ab
    return ans
```
