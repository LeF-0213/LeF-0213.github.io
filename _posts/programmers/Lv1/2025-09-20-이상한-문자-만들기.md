## Java
```java
class Solution {
    public String solution(String s) {
        String answer = "";
        s = s.toLowerCase();
        String[] strArray = s.split("");
        int cnt = 0;
        
        for(String str : strArray) {
            answer += cnt % 2 == 0 ? str.toUpperCase() : str;
            cnt = str.equals(" ") ? 0 : cnt + 1;
        }
        
        return answer;
    }
}
```
