## Java
```java
class Solution {
    public long solution(long n) {
        long answer = 0;
        double x = 0;
        
        x = Math.sqrt(n);
        if(Math.floor(x) == x) {
            answer = (long)Math.pow(x + 1, 2);
        } else {
            answer = -1;
        }
        
        
        return answer;
    }
}
```
