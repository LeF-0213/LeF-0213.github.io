## Java

```java
class Solution {
    public String solution(String s) {
        String answer = s.substring((s.length() - 1) / 2, (s.length() + 2) / 2);
        
        return answer;
    }
}
```

## Python

```python
def solution(s):
    return s[(len(s) - 1) // 2: (len(s) + 2) // 2]
```
