## Java
```java
class Solution {
    public int solution(int[] number) {
        int answer = 0;
        int start = 0;
        int len = number.length;
        
        while(start < len - 2) {
            for(int i = start + 1; i < len - 1; i++) {
                for(int j = i + 1; j < len; j++) {
                    if(number[start] + number[i] + number[j] == 0) {
                        answer++;
                    }
                }
            }
            start++;
        }
        
        return answer;
    }
}
```
