## Java
```java
class Solution {
    public int solution(int[][] sizes) {
        int max = 0, min = 0;

        for(int[] size: sizes) {
            max = Math.max(max, Math.max(size[0], size[1]));
            min = Math.max(min, Math.min(size[0], size[1]));
        }
        
        return max * min;
    }
}
```
