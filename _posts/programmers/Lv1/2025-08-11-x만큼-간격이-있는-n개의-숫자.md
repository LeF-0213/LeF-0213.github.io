## Java
```java
class Solution {
    public long[] solution(int x, int n) {
        long[] answer = new long[n];
        
        for(int i = 0; i < n; i++) {
            answer[i] = (long)x * (i + 1);
        }
        
        return answer;
    }
}
```

## Python
```python
def solution(x, n):
    answer = [i * x for i in range(1, n + 1)]
    return answer
```
