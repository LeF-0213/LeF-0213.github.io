## Java
```java
class Solution {
    public int solution(int n) {
        int answer = 2;
        
        while(n % answer != 1) {
            answer++;
        }
        
        return answer;
    }
}
```

## Python
```python
def solution(n):
    answer = 0
    for i in range(1, n):
        if(n % i == 1):
            answer = i
            break
    
    return answer
```
