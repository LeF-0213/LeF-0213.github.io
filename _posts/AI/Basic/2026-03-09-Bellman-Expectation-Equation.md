# 벨만 기대 방정식 (Bellman Expectation Equation)

벨만 기대 방정식은 어떤 상태의 가치(Value)가                    
**"현재 받을 보상"**과 **"다음 상태의 가치의 기대값"**으로 표현된다는 관계식이다.

지금 이 상태가 얼마나 좋은지는 현재 보상만으로 결정되는 것이 아니라,            
**현재 보상 + 정책에 따라 도달한 다음 상태들의 가치의 평균으로 재귀적으로 정의** 된다.

> "현재 가치 = 지금 얻는 것 + 미래 가치의 평균"

$$V(s) = R + γV(s′)$$

# 벨만 방정식 수식 풀어보기

## 현재만 보기: V(s)≈R
{: .info-box}
가장 단순한 출발점, 지금 받는 보상 하나만 고려하면 상태의 가치는 즉시 보상과 같다.

$$V(s)≈R$$

**V(s)** = 상태 s의 가치 | **R** = 지금 당장 받는 보상

## 한 번, 두 번 앞을 내다 보기
{: .info-box}
미래 보상을 하나씩 더해가되, 멀수록 할인율 γ 를 거듭 곱해 덜 중요하게 반영한다.

$$V(s) = R + γR′$$
$$V(s) = R + γR′ + γ^2R$$

> **각 항의 의미**
> - **R**: 지금 받는 보상
> - **γR′**: 한 칸 뒤 보상 (할인율 γ 한 번 적용)
> - **γR^2R′′**: 두 칸 뒤 보상 (할인율 γ 두 번 적용)
> 
> → 미래로 갈수록 γ 가 거듭 곱해져서 점점 작아진다.

## 무한히 확장하면? → 다시 V(s')로 압축
{: .info-box}
이 패턴을 끝없이 이어가면 무한급수가 된다.              
그런데 V(s')가 이미 그 이후의 모든 미래를 재귀적으로 담고  있기 때문에,             
단 한 줄로 압축하면 벨만 기대 방정식이다.

$$V(s) = R+γR′+ γ2R′′+ γ3R′′′+ ⋯ (무한급수)$$
$$V(s) = R + γV(s′)$$

> **기호 설명**
> - **V(s)**: 현재 상태 s의 가치
> - **R**: 즉시 보상 (지금 당장 받는 것)
> - **γ**: 할인율 (0~1 사이 값), 미래를 얼마나 중시하는 지 결정
> - **V(s')**: 다음 상태 s'의 가치 (같은 구조가 재귀적으로 반복)
>
> - **γ** = 1.0 → 미래 보상을 현재와 똑같이 중요시 (장기적 관점)
> - **γ** = 0.0 → 현재 보상만 고려 (극도의 근시안)
> - **γ** = 0.9 → 한 스텝마다 가치가  10%씩 감소 (일반적 설정)

# 현실 MDP 에서의 벨만 기대 방정식
{: .info-box}
실제 환경에서는 행동이 여러 개이고, 다음 상태가 확률적으로 결정되며,            
정책에 따라 행동 확률이 달라진다.

이를 모두 반영하면 아래 식이 된다.
$$V^{\pi}(s) = \sum_{a} \pi(a|s) \sum_{s'} P(s'|s, a) \left( R(s, a, s') + \gamma V^{\pi}(s') \right)$$

이 식은 아래 4개를 차례로 평균을 내는 구조이다.

|수식 요소|의미|
|-------|--------------------|
|**∑aπ(a∣s)**| 상태 s 에서 정책 $\pi$ 가 고르는 행동드르이 기대값 (행동 평균) |
|**∑s′P(s′∣s,a)**| 행동 a 를 했을 때 나올 수 있는 다음 상태들의 기대값 (전이 평균) |
|**R(s,a,s′)**| 상태 s 에서 행동 a 를 해서 s'에 도달했을 때 받는 즉시 보상 |
|**γVπ(s′)**| | 다음 상태 s'의 가치에 할인율은 적용한 미래 가치 |

> 상태 s에서 정책대로 움직일 때,            
> (즉시 보상 + 할인된 다음 상태 가치) 의 평균이 Vπ(s)

# 실전 예시 - 술 취한 사람 문제

술취한 사람 문제는 강화학습이나 마르코프 과정에서 자주 나오는 아주 대표적인 예제이다.

술에 취한 사람이 1 차원 길 위에 서 있다.                
원하는 방향으로 정확히 움직이지 못하고 확률적으로 비틀거리며 이동한다.          
이때 현재 위치를 상태(State)로 보고,            
다음 위치로 이동하는 확률은 상태 전이 확률로 표현하면 벨만 기대 방정식을 적용할 수 있다.

## 환경 설정

```
[Home(+1)] - A - B - C - [Bar(-1)]
```

|구분|설정|
|------|----------|
|**상태 (S)**| Home, A, B, C, Bar |
|**전이 확률**| 50% 왼쪽 / 50% 오른쪽 |
|**보상 (R)**| Home +1 / Bar -1 / 중간 0 |
|**할인율 (γ)**| γ = 1 |

## 방정식 세우기
{: .info-box}
종료 상태의 가치는 고정: V(Home) = +1, V(Bar) = -1

### 연립 방정식               
- V(A) = 0.5 x V(Home) + 0.5 x V(B) = 0.5 + 0.5xV(B)
- V(B) = 0.5 x V(A) + 0.5 * V(C)
- V(C) = 0.5 x V(B) + 0.5 x V(Bar) = 0.5xV(B) - 0.5

## 연립 방정식 풀기
{: .info-box}
세 방정식을 연립하여 풀면 해석적(analytic) 해를 구할 수 있다.

|상태|가치값|해석|
|-----|-------|-------------|
|**Home**|**+1.0**| 집 도착 → 최고 상태 (종료) |
|**A**|**+0.5**| 집에 더 가까운 비교적 좋은 상태 |
|**B**|**0.0**| 완전 중립 - 집과 술집 사이 정중앙 |
|**C**|**-0.5**| 술집에 더 가까운 좋지 않은 상태 |
|**Bar**|**-1.0**| 술집 도착 → 최악 상태 (종료) |

```
Home(+1) - A(+0.5) - B(0) - C(-0.5) - Bar(-1)
```

## 반복 업데이트 (Iterative)
{: .info-box}
수식을 직접 풀지 않고, A・B・C 를 모두 0으로 초기화한 뒤 반복 대입하는 방식으로도 같은 답에 수렴한다.

```python
V = {
    "Home": 1.0,
    "A": 0.0,
    "B": 0.0,
    "C": 0.0,
    "Bar": -1.0
}

iterations = 10

for i in range(1, iterations + 1):
    u_V = V.copy()
    u_V["A"] = 0.5 * V["Home"] + 0.5 * V["B"]
    u_V["B"] = 0.5 * V["A"] + 0.5 * V["C"]
    u_V["C"] = 0.5 * V["B"] + 0.5 * V["Bar"]
    V = u_V
    print(f"{i:d}회 반복 -> A={V['A']:.4f}, B={V['B']:.4f}, C={V['C']:.4f}")

for state, value in V.items():
    print(f"{state}: {value:.4f}")
```

# 심화 - 5x5 격자 술 취한 사람 문제
{: .info-box}
1차원 길에서 2차원 5x5 격자로 확장한다. 술 취한 사람이 상・하・좌・우 4방향으로 무작위 이동하며, 동일한 벨만 기대 방정식을 반복 업데이트로 풀어낸다.

## 환경 설정

|항목|내용|
|--------|--------------|
|**격자 크기**| 5x5 (총 25개 상태) |
|**Home (0, 0)**| 좌상단 - 도착 보상 + 1, 종료 상태 |
|**Bar (0, 4)**| 우상단 - 도착 보상 -1, 종료 상태 |
|**이동 방향**| 상・하・좌・우 각 25% 확률로 랜덤 이동 |
|**벽 처리**| 격자 밖으로 이동하려 하면 제자리 유지 |
|**할인율 γ**| γ = 1.0 (미래 보상 전부 반영) |
|**반복 횟수**| 30회 (1, 2, 5, 10, 20, 30 회 시점 출력) |

### 🗺️ 격자

```
+-----+-----+-----+-----+-----+
| H   | A   | B   | C   | BAR |
+-----+-----+-----+-----+-----+
| D   | E   | F   | G   | I   |
+-----+-----+-----+-----+-----+
| J   | K   | L   | M   | N   |
+-----+-----+-----+-----+-----+
| O   | P   | Q   | R   | S   |
+-----+-----+-----+-----+-----+
| T   | U   | V   | W   | X   |
+-----+-----+-----+-----+-----+
```

## 벨만 기대 방정식 정용

각 상태에서 4 방향으로 이동할 확률이 동일(각 25%)하므로 수식은 아래와 같다.

$$V(s) = 0.25 x [V(위) + V(아래) + V(왼쪽) + V(오른쪽)]$$

벽에 막히는 방향은 제자리(현재 상태s)로 전이된다.

## 전체 코드

```python
SIZE = 5
GAMMA = 1.0
ITERATIONS = 30
HOME = (0, 0)
BAR = (0, 4)
DIRECTIONS = [(-1, 0), (1, 0), (0, -1), (0, 1)]

def is_terminal(state):
    return state == HOME or state == BAR

def move(state, direction):
    r, c = state
    dr, dc = direction
    nr, nc = r + dr, c + dc

    if nr < 0 or nr >= SIZE or nc < 0 or nc >= SIZE:
        return state
    return (nr, nc)

def get_reward(next_state):
    if next_state == HOME:
        return 1.0
    elif next_state == BAR:
        return -1.0
    else:
        return 0.0

def print_grid(values, title=""):
    if title:
        print(f"\n{title}")

    for r in range(SIZE):
        row = []
        for c in range(SIZE):
            if state == HOME:
                row.append(" HOME ")
            elif state == BAR:
                row.append(" Bar ")
            else:
                row.append(f"{values[r][c]:6.2f}")
        print(" | ".join(row))
    print()

def value_iteration():
    V = [[0.0 for _ in range(SIZE)] for _ in range(SIZE)]

    V[HOME[0]][HOME[1]] = 1.0
    V[BAR[0]][BAR[1]] = -1.0

    print_grid(V, title="초기 상태 가치")

    for iteration in range(1, ITERATIONS + 1):
        new_V = [[V[r][c] for c in range(SIZE)] for r in range(SIZE)]

        for r in range(SIZE):
            for c in range(SIZE):
                state = (r, c)

                if is_terminal(state):
                    continue

                value = 0.0

                for direction in DIRECTIONS:
                    next_state = move(state, direction)
                    reward = get_reward(next_state)

                    nr, nc = next_state
                    value += 0.25 * (reward + GAMMA * V[nr][nc])

                new_V[r][c] = value

        V = new_V

        if iteration in [1, 2, 4, 10, 20, 30]
            print_grid(V, title=f"{iteration}회 반복 후 상태 가치")

    return V

if __name__ == "__main__":
    final_V = value_iteration()

    print_gird(final_V, title="최종 상태 가치")
```
